# Terms of Service Page

**Source:** `frontend/app/terms/page.tsx`
**Route:** `/terms`

Static marketing page rendering the Bwain.app Terms of Service. Client component using `Link` from `next/link`.

## Composition

- Gradient background matching `/privacy`.
- `<Card>`-style wrapper with `prose` styling.
- Sections: Agreement to Terms, Definitions, Account Registration, Acceptable Use, Intellectual Property, Termination, Disclaimer, Liability, Changes, Contact.

## State / Hooks

None.

## Used By

- Footer links on `/` and `/login`.
